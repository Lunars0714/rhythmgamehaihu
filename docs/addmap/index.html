ファイル構成


このゲームの譜面ファイルは ".Exe(実行ファイル)がある階層\MusicGame_Data\StreamingAssets\Music"(以下Musicフォルダ) に格納されています
Musicフォルダに上記の配列に対応したフォルダ(以下譜面フォルダ)があり、その中に譜面ファイル、音楽ファイル、情報ファイルがあります。
譜面フォルダの中にあるファイルを解説します。
Music.wav これは音声ファイルです。wav形式である必要があります。
index.json これは情報ファイルです。json形式で、BPM,bitrate,quantizeの三項目があり、それぞれの情報を入力する必要があります。
Score.json これは譜面ファイルです。json形式で、Setch氏のNoteEditorが使用できます。

実際に作る


2.追記した名前のフォルダをMusicフォルダ直下に作成。
例:musicフォルダの中身が musiclist.json(ファイル) violin(フォルダ) に piano(フォルダ)を追加
3.作成したフォルダの中にNoteEditorで作った譜面をScore.jsonという名前で保存
4.同じフォルダの中に.wav形式の音声ファイルをMusic.wavという名前で保存
5.index.jsonを作成。
形式:{ "BPM":整数, "bitrate":整数, "quantize":整数 }
BPMはそのまま曲のBPMです。
bitrateはMusic.wavのビットレートを入力してください。
quantizeはNoteEditorのLPBを入力してください。
例:{ "BPM":170, "bitrate":44100, "quantize":4 }

これで譜面のリロードをすれば譜面が追加されているはずです、作業お疲れ様です。
もし譜面が読み込まれない場合は、再起動も試してみてください。
